## Get client Data

const mintPubKey = new web3.PublicKey(event.target.mint.value);
        const recipientPubKey = new web3.PublicKey(event.target.recipient.value);

## Create ATA

Now we create `two ata`
for both `sender` and `recipient`

using `same token mint`

```tsx
const senderAta = await getAssociatedTokenAddress(
            mintPubKey,
            publicKey,
            false,
            TOKEN_PROGRAM_ID,
            ASSOCIATED_TOKEN_PROGRAM_ID
        );

        const receiverAta = await getAssociatedTokenAddress(
            mintPubKey,
            recipientPubKey,
            false,
            TOKEN_PROGRAM_ID,
            ASSOCIATED_TOKEN_PROGRAM_ID
        );
```

## ATA Transaction

here next block of code
sending ata creation tx into blockchain (using solana web3js)

```tsx
try {
            await getAccount(connection, receiverAta);
        } catch (e) {
            transaction.add(
                createAssociatedTokenAccountInstruction(
                    publicKey,
                    receiverAta,
                    recipientPubKey,
                    mintPubKey,
                    TOKEN_PROGRAM_ID,
                    ASSOCIATED_TOKEN_PROGRAM_ID
                )
            );
        }
```

## Transfer Rpc Call

Next we create a rpc call to transfer spl token contract methods

```tsx
const transferTx = await program.methods
            .transferTokens(amount)
            .accounts({
                from: senderAta,
                to: receiverAta,
                owner: publicKey,
            })
            .instruction();

        transaction.add(transferTx);

        const signature = await sendTransaction(transaction, connection);

        await connection.confirmTransaction(signature, "confirmed");
```
