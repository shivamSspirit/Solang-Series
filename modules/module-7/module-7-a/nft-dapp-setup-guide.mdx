## Run dapp locally

```bash
git clone git@github.com:shivamSspirit/solang-Nft-mint-transfer.git
```

```bash
cd solang-Nft-mint-transfer
```

```bash
npm install
```

You are now ready to run the DApp and go through the Instructions and transactions.
